Tutzy
Smart Tutor Finding & Booking App (Flutter + Firebase)

Tutzy is a mobile application designed to connect students with tutors seamlessly. It enables users to search, view, and book tutors based on subjects, availability, and time slots.
Built using Flutter and Firebase, the app focuses on real-time booking, smooth user experience, and scalable backend architecture.


Flow:
User Login → Browse Tutors → View Profile → Select Time Slot → Book Session → Confirmation


🏗️ System Pipeline
Logic:
Flutter App → Firebase Authentication → Firestore Database → Booking Engine → UI Update


🚀 Key Features

-   Tutor Search & Filtering
-   Real-Time Slot Booking System
-   Tutor Profiles with Availability
-   Secure Authentication (Firebase)
-   Dynamic UI Updates


🛠️ Tech Stack

-  Flutter (Dart)
-  Firebase Authentication
-  Cloud Firestore (spark plan)
-  Firebase Hosting (spark plan)


⚙️ How It Works

-  User signs up / logs in
-  Browses available tutors
-  Selects preferred date and time slot
-  Booking is validated and stored in database
-  UI updates instantly with confirmation


🎯 Engineering Highlights

-  Real-time slot validation to prevent double booking
-  Efficient state management for dynamic UI
-  Scalable backend using Firebase services
-  Clean separation of UI and business logic


📈 Use Cases

-  Private tutoring platforms
-  Coaching institutes
-  Online learning systems


🔮 Future Improvements

🧠 Vision

To simplify tutor discovery and booking through a fast, scalable, and user-friendly platform.
